# Collatz Conjecture

## Simple implementation of the Collatz conjecture problem

## Algorithm:
input n a positive integer

```
begin Compute (n)
    if n even then
        Compute <- Compute (n / 2)
    else
        Compute <- Compute (3 * n + 1)
    end if
```
